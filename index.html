    • O que é o GIT e o GitHub?



    O GIT é um software de controle de versão, onde conseguimos gerar registros do nosso projeto mantendo o controle em quais arquivos foram alterados, quando foram alterados e por quem.

    O GitHub é uma espécie de rede social de desenvolvedores, é uma plataforma onde você pode subir seus projetos, criar seu portfólio e compartilhá-lo com o mundo.


    Comando do GIT

    Verifica arquivos que foram alterados

    git status

    Prepara um arquivo para ser enviado para o GitHub

    git add nome_de_um_arquivo

    Gera a versão das modificações

    git commit -m "uma mensagem legal do que foi feito nos arquivos nessa versão"

    Envia para o GitHub

    git push origin main
